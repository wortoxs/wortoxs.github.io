创建组件

React 组件必须以大写字母开头，而 HTML 标签则为小写字母。

组件不能返回多个 JSX 标签，必须将其包含在 <div></div> 或 <></> 中。

样式

由于 HTML 中的 class 属性与 JavaScript 中的 class 关键字相冲突，因此 JSX 中使用 className 来表示 CSS 类。

数据展示

使用花括号来使用 JavaScript。

条件渲染

使用 if 语句或 ?: 来条件性的包含 JSX

如果不需要 else 分支的话也可以使用 &&

渲染列表

使用 for 循环或 map() 来渲染组件列表

注意到 li 有 key 属性，对于列表中的每个对象，都应该使用 string 或 number 来唯一标识它们。React 将使用这些 key 来监测之后的插入、删除，重排等操作。
