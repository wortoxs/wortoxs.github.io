从零开始的 React 学习 - Part 3

响应事件

通过向组件的 onClick 属性传递函数，可以对事件进行处理。需要注意的是需要向组件传递函数而非函数调用。

事件传递

除了 onScroll 事件，所有事件默认会向上传递。

若要阻止事件进行向上传递，则可以使用 stopPropagation 方法。

在已经阻止事件进行传播的组件中，若要在父组件中仍对其进行捕获，则可以使用 onClickCapture 属性。

取消组件的默认行为

一些组件中包含特殊的行为，例如在 <form>中，进行表单提交后默认会对页面进行刷新。要取消组件的默认行为，可以在 onClick 属性中使用 preventDefault() 方法。

State

React 要求组件为纯函数，无副作用。对于组件中可能发生的改变，则需要使用 State。
使用 useState() 函数，改函数需要传递一个参数来作为 state 的初始值，并返回两个值，分别为 state 值和更新 state 值的方法。

在 React 中，所有以 use 开头的函数均被称为 Hooks。

useState()

React 在进行渲染时，针对不同的组件，会将组件中使用的 useState() 函数按照次序存到数组中，后续再次对组件进行渲染时，会按照顺序依次进行取出。因此使用 useState() 时，不应当将其在条件语句、循环等语句中使用。而一般将其放在函数的最上层。
